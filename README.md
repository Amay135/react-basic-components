# React Basic Learning

Proyek ini dibuat untuk mempelajari dasar-dasar React menggunakan Vite sebagai build tool.

## Features

- React + Vite setup
- Functional Components
- JSX Syntax
- Fast Hot Module Replacement (HMR)
- Basic React project structure

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS

## Installation

Clone repository:

```bash
git clone https://github.com/Amay135/react-1.git
```

Masuk ke folder project:

```bash
cd react-1
```

Install dependencies:

```bash
npm install
```

Menjalankan project:

```bash
npm run dev
```

Buka browser dan akses:

```text
http://localhost:5173
```
